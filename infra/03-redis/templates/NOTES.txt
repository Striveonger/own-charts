Redis has been deployed successfully.

To connect to Redis:

1. Run a Redis client pod:
   kubectl run redis-client --rm -it --image=redis:7.0.11 --restart=Never -- redis-cli -h {{ include "redis.fullname" . }} -p {{ .Values.service.redisPort }}

2. Or use port-forward:
   kubectl port-forward svc/{{ include "redis.fullname" . }} {{ .Values.service.redisPort }}:{{ .Values.service.redisPort }}

The default password is: {{ .Values.redis.password }}

To check the pod status:
   kubectl get pods -l app.kubernetes.io/name=redis
